180 pontos

O triste fim de uma longa vida, os poucos seres de vida longa que chegam a essa idade lentamente definham ao longo de alguns séculos até que finalmente morrem, mas não sem antes passar seus conhecimentos quase intermináveis.

Essa idade não é válida para Dyns.

Mudanças

Força -6, Agilidade -6, Saúde -6, Fadiga -8, velocidade -40%. Distribua -6 pontos nos dons de Carisma, Intuição, Lógica e Sanidade.

Ao invés de receber 3 pontos para gastar em habilidades para todas as habilidades, receba 21 para os últimos séculos.

Você pode escolher três habilidades para ter um máximo inicial um ponto maior e duas para ter dois pontos.
